*
    * 三角函数
        \def{三角函数}
            单位圆 $x^2 + y^2 = 1$
            $\theta$ 半径线段x与x轴正向所成的夹角
            $
                cos \theta = x
                sin \theta = y
            $
    * 双曲函数
        \def{双曲函数}
            单位双曲线 $x^2 - y^2 = 1$
            $
                cosh \theta = x
                sinh \theta = y
            $

    \Property
        $
            \sin  x = \frac{e^{ix} - e^{-ix}}{2 i}
            \cos  x = \frac{e^{ix} + e^{-ix}}{2}
            \sinh x = \frac{e^{x}  - e^{-x} }{2}
            \cosh x = \frac{e^{x}  + e^{-x} }{2}
        $

* Euclidean几何
    * \bf{公理}
        * 直线公理: 过相异两点，能作且只能作一直线.
        * 任意线段能无限延长成一条直线.
        * 圆公理:   以任一点为圆心、任意长为半径，可作一圆.
        * 角公理:   凡是直角都相等.
        * 平行公理: 在平面上, 过直线外一点, 有且只有一条直线与该直线不相交.
